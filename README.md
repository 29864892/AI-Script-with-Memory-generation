A basic chatbot with persona management, multi-persona conversations, and memory generation. Features may not properly work. Memory generation and refinement were not recently tested and may not work, but the core idea was reviewing previous conversation information and generating memory objects to store in memory files based on categories. Refinement adds vector embeddings to each memory object and removes exact matches and memories with 95%+ similarity. Asks for user verification for less certain matches. The included config file should be used to edit file paths (note that some sections of the code may still have hard coded paths as this was implemented later).
